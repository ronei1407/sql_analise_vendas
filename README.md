# SQL Análise de Vendas (SQLite + Python)

Projeto de portfólio com **SQL (SQLite)** integrado ao **Python** para modelagem de dados e análises de vendas.

## Tecnologias
- Python
- SQLite
- SQL (CREATE, INSERT, SELECT, JOIN, GROUP BY)
- Git/GitHub

## Estrutura
- `scripts/criar_db.py` → cria as tabelas (schema)
- `scripts/inserir_dados.py` → insere dados de exemplo
- `scripts/queries.sql` → consultas SQL
- `scripts/rodar_queries.py` → executa as consultas e exibe os resultados

## Como rodar
```bash
python scripts/criar_db.py
python scripts/inserir_dados.py
python scripts/rodar_queries.py
